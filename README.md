# task-manager

A task management app in MERN (MongoDB, Express, React, Node).

## Technologies

### Back-end

- Node.js
- Express.js
- Mongoose.js

### Front-end

- React.js

### Database

- MongoDB

### Authentication

- JWT

### Deployment

- Heroku
- MongoDB Cloud Atlas (databaase)

## MVP

- Login
- Logout
- Register new user
- Read user details
- Read all tasks
- Create task
- Delete task
- Toggle complete
- Heroku deploy
- CSS

### Stretch Goals

(In order of priority)

- Jest Testing
- Update user details
- Delete user account
