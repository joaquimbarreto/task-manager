# task-manager

A task management app in MERN (MongoDB, Express, React, Node).

## Technologies

### Back-end

- Node.js
- Express.js
- Mongoose.js

### Front-end

- React.js

### Database

- MongoDB

### Authentication

- JWT

### Deployment

- Heroku
- mLab addon

## MVP

- Login
- Logout
- Register new user
- Read all tasks
- Create task
- Delete task
- Heroku deploy

### Stretch Goals

- Read user details
- Update user details
- Delete user account
- Get details of task
- Update task
