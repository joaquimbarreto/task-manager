# task-manager

A task management app in MERN (MongoDB, Express, React, Node).

## Technologies

### Back-end

- Node.js
- Express.js
- Mongoose.js

### Front-end

- React.js

### Database

- MongoDB

### Authentication

- JWT

### Deployment

- Heroku
- mLab addon

## MVP

- Login (ok)
- Logout (ok)
- Register new user
- Read all tasks (ok)
- Create task (ok)
- Delete task (ok)
- Update task
- Heroku deploy

### Stretch Goals

- Read user details
- Update user details
- Delete user account
- Get details of task
